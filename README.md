# KSGame_Unity

## 간단한 FPS 게임    

> W,A,S,D 로 이동하며 마우스가 향한 곳으로 총을 발사하여 적을 최대한 잡는 게임    
> [원본 코드](https://wikidocs.net/18100)

---
## 개선사항    
1. 캐릭터 모델 적용    
2. 캐릭터 애니메이션 적용    
3. 적 캐릭터 모델 적용    
4. 적 캐릭터 애니메이션 적용    
5. 게임 맵 변경    
6. 총알 모델 적용    
7. 크로스헤어 추가    
8. 총알이 화면 중앙을 향해 발사되도록 수정
9. 커서 숨김, 락
